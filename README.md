# angular-graphql
Sample project using Angular and Graphql

### Server GraphQL MySQL Typescript CRUD

### Installation

```
npm run build
npm start
```

for development:

```
npm run dev
```

#### Environment variables

```
DB_HOST=localhost
DB_PORT=3306
DB_NAME=graphql
DB_USERNAME=root
DB_PASSWORD=
```
### FRONT END ANGULAR

```
npm install
ng serve
```
